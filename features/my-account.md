# 我的账号

`我的账号` 是登录后的默认入口。它承担的不是“设置页”那么简单，而是把账号状态、绑定信息和基础自助操作集中在一个地方。

## 页面位置

- 菜单：`我的账号`

## 你能在这里做什么

- 查看用户名、角色和 Emby ID
- 查看当前订阅状态和到期时间
- 查看剩余天数
- 更新邮箱
- 修改密码
- 生成 Telegram 绑定码
- 解除 Telegram 绑定

## 页面主要区域

### 账号卡片

顶部卡片会显示：

- 当前身份
- Emby ID
- 订阅状态
- 有效期到哪一天

如果账号已过期，这里会直接给出明显提示，并引导你去 `续费中心`。

### 基础资料

你可以在这里维护：

- 邮箱
- 登录密码

### Telegram 绑定

如果你还没有绑定 Telegram，可以在这里生成绑定码。生成后：

1. 打开 Bot
2. 发送 `/bind`
3. 按提示完成绑定

如果已经绑定，你也可以从这里主动解绑。

### 媒体信息

在账号状态正常时，页面还会显示媒体库统计等信息。对于已过期的普通用户，这部分可能会降级或不显示。

## 推荐操作步骤

### 第一次登录

1. 先确认用户名和 Emby ID 是否正确
2. 看当前账号是否有效、剩余多少天
3. 补齐邮箱
4. 如果要接 Bot，立刻生成绑定码

### 忘了还有多久到期

1. 打开 `我的账号`
2. 看顶部状态卡
3. 如果快到期，直接跳转去 [续费中心](./payments.md)

## 常见结果与限制

- 登录后的账号都会先进入这个页面
- 已过期用户仍能看到状态和续费入口
- Telegram 绑定可以直接自助完成
