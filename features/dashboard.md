# 概览

`概览` 是普通用户登录后的默认落点。它不负责改资料，而是先把账号状态、媒体规模、服务器入口和常用跳转摆在一起。

## 页面位置

- 侧边栏菜单：`概览`

## 你能在这里做什么

- 查看当前账号是否有效、是否已经过期
- 查看到期时间和剩余天数
- 查看 Telegram 是否已经绑定
- 查看媒体库统计
- 复制 Emby 服务器地址
- 快速跳转到 `账号中心` 和 `订阅管理`

## 页面怎么理解

### 顶部账号卡片

页面最上方会直接显示：

- 当前用户名
- 账号类型
- 到期时间
- Telegram 是否已绑定
- 当前会员状态

如果你已经过期，这里会明确告诉你服务已暂停，并引导你去 [续费中心](./payments.md)。

### 媒体统计

中间的统计卡会展示当前服务器的大致规模：

- 电影数量
- 剧集数量
- 总集数

这部分更适合快速判断库规模，不是搜索页。

### 服务器连接

如果账号状态正常，页面会展示 Emby 服务器地址，并支持一键复制。  
控制台和 Emby 客户端共用同一套账号密码。

如果普通用户已经过期，这一块会变成“访问已锁定”，不会继续展示服务器入口。

### 快捷操作

页面底部会保留最常用的两个跳转：

- 去 [账号中心](./account-center.md)
- 去 [订阅管理](./subscriptions.md)

## 推荐操作步骤

### 第一次登录

1. 先看账号是否有效、到期时间是否正常
2. 确认 Telegram 是否已经绑定
3. 复制服务器地址，在播放器里完成登录
4. 如果需要改邮箱、密码或绑定 Bot，再去 [账号中心](./account-center.md)

### 登录后发现看不了

1. 先回到 `概览`
2. 看页面是否已经提示“服务已暂停”
3. 如果已过期，直接去 [续费中心](./payments.md)

## 常见结果与限制

- `概览` 是状态页，不是设置页
- 普通用户过期后仍然能进控制台，但服务器入口会被锁定
- 复制服务器地址失败时，通常是浏览器权限或系统剪贴板限制，不代表服务器本身有问题
