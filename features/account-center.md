# 账号中心

`账号中心` 负责处理资料、安全和 Telegram 绑定。它不是首页状态总览页，重点是“改”和“绑”。

## 页面位置

- 侧边栏菜单：`账号中心`

## 你能在这里做什么

- 查看用户名、角色和 Emby ID
- 查看当前有效期和创建时间
- 更新联系邮箱
- 修改登录密码
- 生成 Telegram 绑定验证码
- 解除 Telegram 绑定

## 页面主要区域

### 基本信息

这里会展示：

- 用户名
- Emby ID
- 联系邮箱
- 当前有效期
- 创建时间

其中 Emby ID 主要用于核对账号关联是否正确，不是你平时登录用的字段。

### 安全设置

这一块用来修改登录密码。修改成功后：

- Ember 控制台要用新密码重新登录
- Emby 客户端也要改用同一套新密码

如果你只改了网站密码却忘了播放器，通常会表现成“网站能进，播放器登不上”。

### Telegram 绑定

如果还没绑定，页面会先给你生成一个 6 位验证码。  
你需要把它发给 Bot：

```text
/bind 123456
```

如果已经绑定，页面会展示当前 Telegram ID，并允许主动解绑。

## 推荐操作步骤

### 第一次接入 Bot

1. 打开 `账号中心`
2. 生成 Telegram 绑定验证码
3. 去 Bot 私聊发送 `/bind 6位验证码`
4. 绑定成功后再用 `/info`、`/redeem`、`/search`

### 想改密码

1. 在 `安全设置` 里输入当前密码
2. 填新密码并确认
3. 提交后，用新密码重新登录 Ember 和 Emby

## 常见结果与限制

- 绑定验证码是临时的，过期后需要重新生成
- Telegram 绑定和解绑都可以自助完成
- 修改密码后，不会自动帮你更新各个播放器里的已保存密码
