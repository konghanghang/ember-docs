# 用户使用手册

这一组文档不是技术说明，而是控制台功能说明。你可以把它理解成 Ember 的菜单级使用手册。

## 用户侧菜单地图

登录后，普通用户最常见的菜单是：

- [概览](./dashboard.md)
- [账号中心](./account-center.md)
- [我的画像](./profile-analytics.md)
- [订阅管理](./subscriptions.md)
- [播放排行榜](./rankings.md)
- [媒体库](./library.md)
- [追剧日历](./tv-calendar.md)
- [续费中心](./payments.md)

如果你还没用过控制台，先看 [控制台总览](./user-console.md)。

## 推荐阅读顺序

### 我是第一次登录

1. [概览](./dashboard.md)
2. [账号中心](./account-center.md)
3. [续费中心](./payments.md)
4. [Telegram Bot](./telegram-bot.md)

### 我还没账号，或者忘了密码

1. [登录、注册与找回密码](./account-access.md)
2. 再回来看 [第一次使用](../getting-started.md)

### 我想知道装哪个播放器最省事

1. [播放器推荐](./players.md)
2. 再按自己的设备去对应小节看接入步骤

### 我想找内容、提需求

1. [媒体库](./library.md)
2. [播放排行榜](./rankings.md)
3. [订阅管理](./subscriptions.md)
4. [追剧日历](./tv-calendar.md)

## 先理解两个事实

- Ember 的主入口是统一控制台。
- 过期用户仍然能看到概览、账号中心和续费入口，但服务器访问等能力会降级。
