# 控制台总览

Ember 的网站主要围绕统一控制台展开。登录后，你的大部分操作都会从侧边栏菜单进入。

## 普通用户会看到什么

侧边栏通常包含这些菜单：

- `概览`
- `账号中心`
- `我的画像`
- `订阅管理`
- `播放排行榜`
- `媒体库`
- `追剧日历`
- `续费中心`

## 推荐使用顺序

### 第一次登录

1. 先进 [概览](./dashboard.md) 确认账号状态、到期时间和服务器入口
2. 再去 [账号中心](./account-center.md) 处理邮箱、密码和 Telegram 绑定
3. 如果账号快到期或已过期，直接去 [续费中心](./payments.md)

### 日常使用

1. 想看最近内容：进 [媒体库](./library.md)
2. 想知道最近大家在看什么：进 [播放排行榜](./rankings.md)
3. 想申请想看的片：进 [订阅管理](./subscriptions.md)
4. 想追连载：进 [追剧日历](./tv-calendar.md)
5. 想了解自己的播放习惯：进 [我的画像](./profile-analytics.md)

## 到期后会发生什么

Ember 不会把过期用户直接踢出整个控制台。你仍然可以：

- 查看当前账号状态和剩余信息
- 进入账号中心修改资料或重新绑定 Bot
- 进入续费中心
- 获取必要的上下文信息

但对于普通用户，像服务器地址这类入口会在过期后被锁定。  
这样做的目的很直接：让你知道自己为什么不能继续访问，以及下一步该去哪里恢复服务。
