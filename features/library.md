# 媒体库

`媒体库` 用来浏览 Emby 最近入库的内容，适合回答“最近上了什么”和“有没有新片/新剧”。

## 页面位置

- 菜单：`媒体库`

## 你能在这里做什么

- 查看最近入库的电影
- 查看最近入库的剧集
- 刷新当前列表
- 通过海报快速判断是否值得去 Emby 里继续看

## 页面怎么理解

### 类型切换

页面顶部可以在两种类型之间切换：

- `电影`
- `剧集`

切换后，列表会重新读取对应的最近入库内容。

### 内容卡片

每张卡片通常会展示：

- 海报
- 标题
- 年份
- 评分
- 分级

剧集还会显示子集数之类的简要信息。

## 推荐操作步骤

### 想看看最近加了什么

1. 打开 `媒体库`
2. 先看电影还是剧集
3. 需要时点击刷新
4. 找到感兴趣的内容后，再去 Emby 里继续浏览和播放

## 常见结果与限制

- 这个页面展示的是“最近入库”，不是完整 Emby 全量目录
- 海报偶尔可能加载失败，页面会回退到占位图
- 这页更适合发现内容，不适合做复杂搜索
